# Riddle Me This

A Media Collection Application Experiment

# Summary
This was an web application I wrote to teach myself more about creating websites from scratch and how to use AWS.

# Installation

```sh
npm i
```

# Running 

```sh
ng serve --open
```

```sh
ng test
```
